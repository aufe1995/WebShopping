# WebShopping
이 프로젝트는 웹 쇼핑몰의 관리자 페이지를 제작하는 프로젝트 입니다.

사용한 기술 : JSP, JAVA, Mysql

프로젝트 설명 : 이 프로젝트는 회원가입 및 로그인, 관리자 페이지 브랜드, 물품을 추가 및 수정하는 웹 페이지입니다.
